Was playing with ssd1306xled library from - The Tinusaur Project. This is ssd1306 compatible oled display hooked to atmel's attiny85 mcu, I used vector images -  
"FREE CHRISTMAS VECTOR ICONS"   
from http://sivioco.com/blog/free-christmas-vector-icons cropped and scaled them to appropriate size, thresholded and exported them to bitmap format, then transformed to array of hex data in form of c header file using  
http://en.radzio.dxp.pl/bitmap_converter ssd1306xled library already provides some test examples which can be easily modified for your needs.
