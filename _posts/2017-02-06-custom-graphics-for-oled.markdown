Just before winter holidays I wrote a post about assembling a simple prototype for electronic greeting. Christmas time is long gone, but I wanted to share how to create graphics for OLED display unit. As usually I do all my work on linux based computer operating system. This time was no exception as I am running Debian 8.6 codenamed "jessie". As of writing this there already exists version 8.7, I have not upgraded yet.

List of software and items that will be needed

- GNU Image Manipulation Program (GIMP), [https://www.gimp.org/](https://www.gimp.org/){:target="_blank"}
- Wine - a compatibility layer capable of running Windows applications on Linux, [https://www.winehq.org](https://www.winehq.org){:target="_blank"}
- LCD Assistant, [http://en.radzio.dxp.pl/bitmap_converter](http://en.radzio.dxp.pl/bitmap_converter){:target="_blank"}, by Radosław Kwiecień
- [https://material.io/icons](https://material.io/icons/){:target="_blank"} - material design icons from Google Design, Google Design is a cooperative effort led by a group of designers, writers, and developers at Google
