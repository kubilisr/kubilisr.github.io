Just before winter holidays I wrote a post about assembling a simple prototype for electronic greeting. Christmas time is long gone, but I wanted to share how to create graphics for OLED display unit. As usually I do all my work on linux based computer operating system - Debian Linux distribution in particular. This time is not an exception as I am running Debian 8.6 codenamed "jessie". As of writing this there already exists version 8.7, I have not upgraded yet.  

List of software and items that will be needed

- GNU Image Manipulation Program (GIMP)
- Wine - a compatibility layer capable of running Windows applications on Linux
- LCD Assistant
- https://material.io/icons/ - material design icons
